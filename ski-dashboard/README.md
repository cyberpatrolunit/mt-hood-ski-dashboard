# Ski Dashboard

A real-time ski conditions dashboard for Mt. Hood Meadows.

## Features

- Live weather conditions and forecasts
- Snowfall history and predictions
- NOAA weather alerts
- Road conditions
- Gear checklist
- Interactive snow visualization
- Taylor Swift MIDI background music

## Setup

```bash
npm install
node server.js
```

Access at `http://localhost:18791`

## Tech Stack

- Node.js + Express
- Chart.js for visualizations
- NWS, Open-Meteo APIs for weather data
- Tone.js for MIDI playback
